Trabajo de Grado: Plataforma Web Automatizada para Biblioteca

Diseñé e implementé una plataforma completa para la gestión bibliotecaria, con múltiples niveles administrativos, automatización de reportes y notificaciones por correo. Pensada tanto para usuarios como para bibliotecarios, su interfaz mejora la experiencia y eficiencia operativa. Este proyecto refleja mi compromiso con soluciones tecnológicas funcionales y escalables. Aunque no está disponible en línea, incluí un carrusel de imágenes que muestra sus principales funcionalidades.
